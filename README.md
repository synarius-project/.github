# Organization profile (`.github`)

This content is meant to live in a GitHub repository named **`.github`** under the **synarius-project** organization.

- **GitHub:** create the repo `synarius-project/.github` (name must be exactly `.github`).
- Push the contents of this directory to the default branch (e.g. `main`).
- The file **`profile/README.md`** is shown on the organization’s public GitHub profile.

Do not confuse this with per-repository `.github/` folders (issue templates, workflows) inside **synarius-core**, **synarius-apps**, or **synarius-studio**.
