# Synarius

**Synarius is an open-source Python toolkit for modeling systems, running simulations, and working with measurement data**—with a visual desktop IDE when you want a GUI, and libraries you can embed or script when you do not.

## What you can do with it

- **Model and wire systems** in a desktop IDE: blocks, connectors, projects, and experiment setup — see **[Synarius Studio](https://github.com/synarius-project/synarius-studio)**.
- **Run and extend simulation logic** from Python without shipping a GUI: projects, FMUs, stimulation, measurements, and data export — see **[synarius-core](https://github.com/synarius-project/synarius-core)**.
- **Inspect time-series and compare calibration data** with focused desktop tools — see **[synarius-apps](https://github.com/synarius-project/synarius-apps)**.

## Why Synarius exists

Building a simulation workflow usually means gluing together a model editor, a solver or co-simulation stack, file formats for parameters and logs, and one-off scripts. **Synarius aims for one coherent Python-centric path**: a desktop IDE for everyday work, a **GUI-free core** for automation and services, and **small apps** for repeated tasks like plotting channels or diffing calibrations.
