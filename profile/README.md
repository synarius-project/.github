# Synarius

Open tools for **graphical system modeling and simulation**, with a clear split between backend, desktop utilities, and the full IDE.

## Programming guidelines

**[Browse the HTML documentation](https://synarius-project.github.io/synarius-guidelines/)** (built from [synarius-project/synarius-guidelines](https://github.com/synarius-project/synarius-guidelines)) — start on the home page, then open **Programming guidelines**.

## Repositories

| | |
|--|--|
| **[synarius-core](https://github.com/synarius-project/synarius-core)** | GUI-less simulation backend (no Qt dependency). |
| **[synarius-apps](https://github.com/synarius-project/synarius-apps)** | DataViewer, ParaWiz, and shared Qt tools — useful **without** installing Synarius Studio. |
| **[synarius-studio](https://github.com/synarius-project/synarius-studio)** | PySide6 IDE for modeling and simulation on top of core + apps. |
